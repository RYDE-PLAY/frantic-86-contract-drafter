# Frantic #86 Contract Drafter Evidence

This repository contains public, reproducible evidence for the
`ryde-play/contract-drafter` runx skill submitted to Frantic bounty #86.

The contract template and parties are synthetic demonstration data. They are
published to exercise the real skill runtime without exposing private legal or
personal information. The resulting document is nonbinding and is not legal
advice.

The final evidence packet will include the immutable dogfood input, runtime
output, runtime-persisted post-publish receipt, public verification key,
`runx verify` result, registry and harness results, and a human-readable report.
